5. API Reference | IFMODStudioModule

This class inherits from IModuleInterface.

Methods:

  • IFMODStudioModule::Get Singleton-like access to this module's interface. Beware of calling this during the shutdown phase, though. Your module might have been unloaded already.
  • IFMODStudioModule::IsAvailable Checks to see if this module is loaded and ready. It is only valid to call Get() if IsAvailable() returns true.
  • IFMODStudioModule::GetStudioSystem Get a pointer to the runtime studio system (only valid in-game or in PIE).
  • IFMODStudioModule::SetSystemPaused Set system paused.
  • IFMODStudioModule::RefreshSettings Called when user changes any studio settings.
  • IFMODStudioModule::SetInPIE Called when we enter or leave PIE mode.
  • IFMODStudioModule::FindAssetByName Look up an asset given its name.
  • IFMODStudioModule::FindEventByName Look up an event given its name.
  • IFMODStudioModule::GetBankPath Get the disk path for a bank asset.
  • IFMODStudioModule::GetAllBankPaths Get the disk paths for all Banks.
  • IFMODStudioModule::ReloadBanks Called by the editor module when banks have been modified on disk.
  • IFMODStudioModule::GetLocale Get current locale.
  • IFMODStudioModule::SetLocale Set active locale. Locale must be the locale name of one of the configured project locales.
  • IFMODStudioModule::GetEventDescription Get an event description. The system type can control which Studio system to use, or leave it as System_Max for it to choose automatically.
  • IFMODStudioModule::CreateAuditioningInstance Create a single auditioning instance using the auditioning system.
  • IFMODStudioModule::StopAuditioningInstance Stop any auditioning instance.
  • IFMODStudioModule::HasListenerMoved Return whether the listener(s) have moved.
  • IFMODStudioModule::SetListenerPosition Used to update a listener's position.
  • IFMODStudioModule::FinishSetListenerPosition Used to update the listener positions.
  • IFMODStudioModule::GetNearestListener Return the audio settings for the listener nearest the given location.
  • IFMODStudioModule::GetFailedBankLoads Return a list of banks that failed to load due to an error.
  • IFMODStudioModule::BanksReloadedEvent This event is fired after all banks were reloaded.
  • IFMODStudioModule::GetRequiredPlugins Return a list of plugins that appear to be needed.
  • IFMODStudioModule::AddRequiredPlugin Register a plugin that is required.
  • IFMODStudioModule::UseSound Returns whether sound is enabled for the game.
  • IFMODStudioModule::LoadPlugin Attempts to load a plugin by name.
  • IFMODStudioModule::LogError Log an FMOD error.
  • IFMODStudioEditorModule::AreBanksLoaded This returns true if the plugin has loaded all required banks on startup. Depending if bLoadAllBanks is checked, this could be all banks or just the Master and Strings Bank.
